🛠️ Troubleshooting Common Issues
================================

Issue 1: Permission Denied When Modifying Parameters
----------------------------------------------------

-bash-4.2$ sudo su -
📤 Output
[root@alnafi-lab ~]#

[root@alnafi-lab ~]# echo 10 > /proc/sys/vm/swappiness
📤 Output

✅ Important reality: `sudo echo 10 > /proc/...` often fails because redirection happens in the non-root shell.
Using `sudo su -` (or `sudo tee`) is the proper fix.

Issue 2: Changes Not Persisting After Reboot
--------------------------------------------

[root@alnafi-lab ~]# nano /etc/sysctl.d/99-custom-tuning.conf
📤 Output

Issue 3: Invalid Parameter Values
---------------------------------

[root@alnafi-lab ~]# sysctl -a | grep parameter_name
📤 Output
(No matching parameter because "parameter_name" is a placeholder—this is exactly how it looks in real labs.)
