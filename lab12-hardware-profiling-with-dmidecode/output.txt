Usage: dmidecode [OPTION]
Options:
 -d, --dev-mem FILE     Read memory from device FILE (default: /dev/mem)
 -h, --help             Display this help text and exit
 -q, --quiet            Less verbose output
 -s, --string KEYWORD   Only display the value of the given DMI string
 -t, --type TYPE        Only display the entries of given type
     --type LIST        List available types
 -H, --handle HANDLE    Only display the entry at given handle
 -u, --dump             Do not decode, dump the raw data
     --dump-bin FILE    Dump the raw data to FILE
     --from-dump FILE   Read the raw data from FILE
 -V, --version          Display version and exit
